# Starter Kits

This folder contains starter kits and templates for building on Hyperliquid.

## Available Starter Kits

### Hyperliquid Explorer

A comprehensive blockchain explorer and monitoring dashboard for the Hyperliquid network, powered by Lava Network's high-performance RPC API.

**Features:**
- Real-time onchain transaction monitoring
- Transaction and address lookup
- Network statistics and analytics
- Interactive charts and visualizations
- Mainnet/Testnet network switching

**Quick Start:**

1. Navigate to the `hyperliquid-explorer` folder
2. Follow the setup instructions in the README.md
3. Get your Lava Network RPC URLs from [https://accounts.lavanet.xyz/](https://accounts.lavanet.xyz/)
4. Add the URLs to `.env.local`
5. Run `npm install && npm run dev`

For detailed setup instructions, see the [README.md](./hyperliquid-explorer/README.md) in the hyperliquid-explorer folder.
